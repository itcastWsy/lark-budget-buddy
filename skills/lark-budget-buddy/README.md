## 这是什么？

lark-budget-buddy 是一个基于 **飞书 CLI** 的 AI Agent Skill，让你通过自然语言和 AI 对话就能完成记账全流程：

```
你：午饭 35 微信
AI：已记录：支出 ¥35.00（餐饮 - 午饭 - 微信）✓

你：本月花了多少？
AI：本月总支出 ¥3,286.50，共 47 笔。餐饮占比最高（36.5%）...

你：生成月度财报
AI：已生成并创建飞书文档 → [点击查看月度财报]
```

**数据存储在你自己的飞书多维表格中**——云端同步、多端可用、不怕丢失。

## 核心功能

### 1. 自然语言记账

说人话就能记账，AI 自动解析金额、分类、支付方式：

| 你说 | AI 理解 |
|------|--------|
| 午饭 35 | 支出 ¥35 / 餐饮 |
| 打车 45 支付宝 | 支出 ¥45 / 交通 / 支付宝 |
| 收入 15000 工资 | 收入 ¥15,000 / 工资 |
| 昨天看电影 80 | 支出 ¥80 / 娱乐 / 昨天 |
| 午饭35，地铁4，咖啡18 | 批量记录 3 笔 |

### 2. 消费查询与统计

- "今天花了多少" → 今日消费总额
- "本月餐饮花了多少" → 按分类筛选统计
- "这周消费明细" → 查看详细记录
- "本月各分类消费占比" → 完整分类统计

### 3. 预算管理

- "设置餐饮预算 2000" → 设定分类月预算
- "本月预算还剩多少" → 各分类预算使用情况
- 每次记账自动检查预算，超支时立即提醒

### 4. 月度财报

一句"生成月度财报"，自动：
- 查询本月所有交易数据
- 生成分类统计、支付方式分布、预算执行报告
- 创建格式化的**飞书文档**（含表格和分析）
- 数据同步展示在**多维表格仪表盘**中

## 快速开始

### 前置要求

- 已安装 [lark-cli](https://github.com/larksuite/cli)
- 已配置飞书应用并完成认证

### 安装 Skill

将 `skills/lark-budget-buddy` 目录复制到你的 Skill 目录：

```bash
# 全局安装
cp -r skills/lark-budget-buddy ~/.agents/skills/

# 或项目级安装
cp -r skills/lark-budget-buddy .agents/skills/
```

### 认证

```bash
lark-cli auth login --domain base,docs,im
```

### 首次使用

对 AI 说：**"帮我初始化记账本"**

AI 会自动创建飞书多维表格，包含"交易记录"和"预算设置"两张表，并返回配置信息。

然后就可以开始记账了！

## 数据结构

### 交易记录表

| 字段 | 类型 | 说明 |
|------|------|------|
| 日期 | 日期时间 | 交易时间（默认当前时间） |
| 类型 | 单选 | 支出 / 收入 |
| 金额 | 货币(CNY) | 交易金额 |
| 分类 | 单选 | 餐饮/交通/购物/娱乐/居住/医疗/教育/通讯/日用/工资/奖金/... |
| 备注 | 文本 | 交易描述 |
| 支付方式 | 单选 | 微信/支付宝/现金/银行卡/信用卡 |

### 预算设置表

| 字段 | 类型 | 说明 |
|------|------|------|
| 分类 | 单选 | 支出分类 |
| 月预算 | 货币(CNY) | 该分类的每月预算上限 |

## 技术架构

```
用户 ─► AI Agent (Claude/Cursor/OpenClaw)
               │
               ├─ 解析自然语言 → 结构化数据
               │
               ├─ lark-cli base +record-upsert  → 写入交易记录
               ├─ lark-cli base +data-query     → 聚合统计
               ├─ lark-cli base +record-list    → 查询明细
               ├─ lark-cli docs +create         → 生成财报文档
               └─ lark-cli im +messages-send    → 推送提醒（可选）
               │
               ▼
         飞书多维表格 (云端存储)
         ├── 交易记录表
         ├── 预算设置表
         └── 仪表盘（可视化）
```

## 与现有方案的区别

| 维度 | 传统记账 App | 本地 JSON 记账 | Budget Buddy |
|------|------------|--------------|-------------|
| 记账方式 | 打开 App → 选分类 → 输金额 | 命令行输入 | 自然语言对话 |
| 数据存储 | App 服务器 | 本地文件 | 飞书多维表格（云端） |
| 多端同步 | 依赖 App | 不支持 | 飞书全平台 |
| 数据分析 | App 内置图表 | 手动分析 | AI 分析 + 多维表格仪表盘 |
| 协作记账 | 部分支持 | 不支持 | 飞书原生协作 |
| 数据导出 | 受限 | 自由 | 自由（飞书支持导出） |
| 扩展性 | 封闭 | 需编码 | Skill 可自由扩展 |

## 常见问题

**Q: 数据安全吗？**
A: 数据存储在你自己的飞书多维表格中，遵循飞书的数据安全策略，不经过任何第三方。

**Q: 支持多币种吗？**
A: 当前默认使用人民币(CNY)。如需其他币种，可在初始化时修改字段的 `currency_code`。

**Q: 可以和家人一起用吗？**
A: 可以。将多维表格分享给家人，设置协作权限即可。每个人通过各自的 AI Agent 记账，数据自动汇总。

**Q: 如何备份数据？**
A: 飞书多维表格支持导出为 Excel/CSV。也可通过 `lark-cli base +record-list --page-all` 导出全部数据。
